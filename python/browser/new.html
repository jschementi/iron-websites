IronPython

Write browser applications in Python

IronPython allows you to use python code in browser applications to script the HTML page and vector-graphics.

IronPython combines the benefits of expressive languages and rich-internet applications with the way web-developer's work. 

<html>
  <head>
    <script type="text/javascript"
      src="http://ironpython.net/d1/dlr-20091120.js"
    ></script>
  </head>
  <body>
    <script type="text/python">
      window.Alert("Hello World from Python")
    </script>
  </body>
</html>
